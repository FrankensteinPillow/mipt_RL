# Управление проектом и зависимостями
Репозиторий организован как python проект с использованием утилиты `uv` для управления зависимостями, их утсановкой, удалением и обновлением. Запуск различных решений происходит так же с использованием `uv`.

Как правило запуски скриптов домашних заданий начинаются с `uv run ...`. Для каждого домашнего задания есть своя директория. В каждом задании есть `README.md` с описанием задания, описанием решения и выводами. Так же в каждом из `README.md` есть пояснения как запускать решение, какие параметры присутствуют и пр. При наличии артефактов обучения моделей, они сохраняются в директории с домашним заданием в директории `./artifacts`.
